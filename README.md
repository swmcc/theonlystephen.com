# theonlystephen.com

Personal CV/portfolio site built with Astro and Tailwind CSS.

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS](https://tailwindcss.com/) v4 - Styling
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
